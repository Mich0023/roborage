# ROBORAGE 2026

## ¿Qué es y para qué sirve?
**ROBORAGE** es la Single Page Application (SPA) oficial desarrollada para el evento de robótica organizado por la división de robótica del capítulo estudiantil **AAAIMX**. 

La aplicación fue diseñada con una estética **Cyberpunk Dark** para reflejar la temática agresiva y futurista de la competencia. Su principal utilidad es brindar a los participantes y al público en general toda la información crítica del evento en un solo lugar:
- **Cuenta regresiva** precisa hacia el día del evento (29 de Abril de 2026).
- **Detalles de las categorías** de competencia (Minisumo Sénior y Batalla de Robots de 1LB, 3LB y 12LB).
- **Ubicación exacta** con un mapa interactivo (Instituto Tecnológico de Mérida).
- **Acceso directo al reglamento** oficial de las competencias.

## Tecnologías Principales
- **React.js** (vía Vite)
- **Tailwind CSS** para los estilos y la variante Cyberpunk.
- **Framer Motion** para animaciones interactivas, fondo de partículas y scroll suave.

## 🚀 Cómo clonar y ejecutar el proyecto localmente

### 1. Prerrequisitos
Asegúrate de tener instalados:
- [Node.js](https://nodejs.org/) (Versión 16+ recomendada)
- [Git](https://git-scm.com/)

### 2. Clonar el Repositorio
Abre tu terminal y ejecuta el siguiente comando para clonar el código a tu máquina local:
```bash
git clone https://github.com/Mich0023/roborage.git
```

### 3. Entrar a la carpeta del proyecto
```bash
cd roborage
```

### 4. Instalar las dependencias
Instala todos los paquetes necesarios de NPM (incluyendo Tailwind CSS y Framer Motion) ejecutando:
```bash
npm install
```

### 5. Ejecutar la Aplicación en Modo de Desarrollo
Una vez instaladas las dependencias, lanza el servidor de desarrollo local:
```bash
npm run dev
```

La terminal te mostrará un enlace (usualmente `http://localhost:5173/`). Solo haz clic o cópialo en tu navegador para visualizar la aplicación en tiempo real.
